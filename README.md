# Tesseracs — html template

Starter project for [Tesseracs](https://github.com/tesseracs) chat sessions.

Clone URL: `https://github.com/tesseracs/templates.html`

## Layout

- `index.html` — loads `styles.css` and `app.js` (ES module).
- `app.js` — imports `greeter.js` and updates the DOM.
- `greeter.js` — small ES module class example.
- `run.sh` — optional local static server (`python3 -m http.server`); use session preview or a browser.
